HTML Validations

They help us to make sure the input from user matches a certain pattern.

There are many built in validations in HTML, for example the email input type would make sure that there is a @ sign in the text.

Also it is worth talking about the required attribute, if entered in the input field, it will not let us submit a form until properly filled out, and can be applied to any input you would logically want to use it on.

Here you can submit the form without even entering text.

But here we used required, so it will have to be filled before submitting. It will show an alert saying to fill the field first.

Of course we can add things to check for specific patterns, like making sure it is between x and x characters(min and max), that it included a special character,etc.

Here we show the input type email with required on, a message tells us to make sure we enter the @ sign if we have not.
