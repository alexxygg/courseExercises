Inheritance

This means that when we are not specific when styling, the elements inside whatever we styled will inherit those values.

Here we have a form with 3 buttons inside it, each with its own label.

Not the form, labels or buttons were styled, only the body. Since all of these elements are inside the body, they have inherited the color darkgreen from it.

This was all the code: body { color: darkgreen; }

This paragraph is inside a div, which we styled with this code: div p { color: darkblue; }

The rest of the content is still dark green, but not that one!

There is a property we can set for any element, class or id called inherit, it inherits the styles of the nearest parent.

Not every element will inherit every property, and not all properties are inheritable.
