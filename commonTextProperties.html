Common Text Properties

There are many properties used for our text, here are a few.

Text-align

Text align helps us align the text to the center, left (which is the default), right and justify. Those are at least the most common uses.

The title has been text aligned to the center. The header for this property was aligned to the right.

Font-weight

This attribute helps us make the font lighter or bolder.

We can enter a value from 100 to 900 if the font allows it, only going up or down between these levels in steps of 100.

A value given of 400 would be the normal font and 700 would be bold.

We can also enter lighter, normal, bold and bolder as options

The header for this attribute was set to a value of 900.

Text-decoration

This attribute helps us underline, overline, cross out and more, but we can also select a color for it directly from the attribute and choose a style for the line.

The line can be regular, dotted, wavy and many more.

Below we used a red wavy line-through.

This text has a line through it.
Letter-spacing

This one just spaces out the letters, like we did with this header.

We entered a value of 15px, but it can be any other too.

Line-height

Line-height just separates lines, increasing or decreasing the space between them.

The paragraph below has a line-height of 50px.
