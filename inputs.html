Inputs

A form is a document section containing interactive controls for submitting information.

The action attribute comes with it, this is to enter the location where the info will be sent to.

The input element is a self-closing one.

Below are four input types, text, color, password and number.

The placeholder attribute is used to have a grayed out text shown to the user, which is immediately removed upon the user entering any information. It is inside the input element.

Text is used for usernames, or any other text.

Color is to choose any color. No placeholder for color unfortunately.

Password is pretty starightforward, you can not see any text entered for security purposes.

Number is for any number, all keys that are not numbers will not work with this input type.

Also it has an up and a down arrow to decrease or increase the number entered.
