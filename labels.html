Labels

Labels are used to associate an input to a text. They connect and prompt you to type when clicked on.

We also need to give the input an ID to link them, making it easier to style them with CSS in the fiture if needed.

The prompt text is entered between the label elements, but outside of the input. The placeholders can be different.
