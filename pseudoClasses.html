Pseudo clasess

These are action classes, only applied when an action is being performed.

They have a colon and a space before the word.

This link has the : active feature styled. It will look red when clicked before being released.

Active link

The first-child class helps only select the first child in a group

Here we have an unordered list, only the first list item has a red background.

Here is the code: ul :first-child { background-color: red; }

The hover class helps us style an element as it will appear when the user hovers over it.

Here we have a button, the text changes to gold color and the background-color to pink when hovered!

The nth-child class helps choose only a particular child by using the number it would have in order.

Here we have many buttons, but only the fourth one will have a different background-color.

Here is the code: .childrenList :nth-child(4) { background-color: aqua; }

We can also select every 2nd child, or every 5th, etc. with nth-of-type.

Here is the code: .secondChildrenList :nth-of-type(3n) { background-color: gold; }, we are selecting the element at every index 3.

Or odd and even only!

.odd :nth-of-type(odd) { background-color: aqua; }

.even :nth-of-type(even) { background-color: darkblue; }
