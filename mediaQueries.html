Media Queries

They allow us to modify our content when certain parameters are met with values like width, device type.

Exact

@media (width: 300px) { h1 { color: blue; } }

This is the syntax, and it means the color of the h1 header text will change to blue when the width is exactly 300px.

Here, the h1 text color will be salmon as soon as the width is 300px or more.

Order

If we want to give a page multiple max or min widths we have to start with the biggest and finish with the smallest, if else, it will cascade and only apply the last rule, since the parameters are the same, and it will only take into account the last value.

Here it is set to the colors of the rainbow, but in reverse order when styling the CSS.

@media (max-width: 800px) { #rainbow { color: purple; } } @media (max-width: 700px) { #rainbow { color: blue; } } @media (max-width: 600px) { #rainbow { color: green; } } @media (max-width: 500px) { #rainbow { color: yellow; } } @media (max-width: 400px) { #rainbow { color: orange; } } @media (max-width: 300px) { #rainbow { color: red; } }
