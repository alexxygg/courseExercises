Other not too frequent elements and tips?

The range input type is a slider, it can be used to determine music level, etc.

Make sure the label for entry and input id and name match to sync these two.

The min and max can be adjusted as needed.

The step option allows us to go up or down by that amount only.

Here we chose step to 10, so no number selection will end in 1-9.

The number input from a while back also takes min and max, so it will not accept less than 1 or more than 1000 in this next example.
