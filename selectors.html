Selectors

Selectors help us style our HTML.

Universal selector *

The universal selector selects every single element on the page and styles it according to what we enter.

We used it for the text color, which includes that of the buttons.

The code looks like this: * { color: blueviolet; }

The element selector

This one uses the element we enter only, such as h1,h2,button,etc.

The background color of the h2 elements was styled this way.

This was the code used: h2 { background-color: aquamarine; }

Combining element selectors

We can also list multiple element selectors with a comma in between them to have the styles applied to all of them specifically.

We styled the h1 and paragraphs background colors this way.

This was the code used: h1, p { background-color: gold; }
